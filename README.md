# Benchmarking and Fine-tuning Computer Vision Models for Object Detection Tasks

This repository contains the research code for Benchmarking and Fine-tuning Computer Vision Models for Object Detection Tasks

## Abstract
This paper benchmarks several computer vision models and fine-tuning approaches for object detection tasks across diverse datasets in order to understand how various architectures and fine-tuning strategies impact performance on specialized tasks relating to object detection and classification in field settings. We evaluate both zero-shot and fine-tuned performance on datasets from different domains, including underwater imagery and tactical surveillance-style data. Of the models fine-tuned to the data, the YOLOv8 architecture showed the most promise with respect to accuracy in classification.
